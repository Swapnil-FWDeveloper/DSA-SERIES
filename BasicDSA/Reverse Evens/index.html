Reverse Evens

Description

Given an arrayAofNpositive integers, print the Even integers in reverse order.

Input

The first line of input contains a single integerN, number of integers.

The second line of input containsNspace-separated integers ,A[0], A[1], .., A[N-1]

Output

The first line of output contains a single integerK, number of even integers.

The second line of output containsKspace-separated even integers in reverse order.

Sample Input 1

5

1 2 3 4 5

Sample Output 1

2

4 2
